Hide

Name:

.hide()

Description:

Hide the matched elements.

Syntax:

$(selector).hide()

Example:

$( "#clickme" ).click(function() {
                            $( "#book" ).hide( "slow", function() {
                            alert( "Animation complete." );
                            });
                        });
